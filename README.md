# Personal-Learn-Path 整理自己工作中用到的一些技术

### 日志处理：  
  **Elmah** : 小型的日志收集查看系统，只能记录一段时间的日志  
  **Splunk** : 商业系统，功能完善  
  **Log4net**: 记录文件，数据库日志  
  **NLog**：同Log4net一样的功能  
  **ELK**：Elasticsearch,logstash,kibana ELK全家桶，开源免费，功能强大  
  **Exceptionless**： 基于ELK实现的
